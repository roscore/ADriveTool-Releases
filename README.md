# ADriveTool Releases

**A-Robot Motor Controller Configuration Tool** 공식 릴리스 저장소입니다.

## 다운로드

최신 버전은 [Releases](https://github.com/roscore/ADriveTool-Releases/releases) 페이지에서 다운로드하실 수 있습니다.

## 설치 방법

1. [Releases](https://github.com/roscore/ADriveTool-Releases/releases) 페이지에서 최신 ZIP 파일 다운로드
2. 원하는 위치에 압축 해제
3. `ADriveTool.exe` 실행

## 시스템 요구사항

- **운영체제**: Windows 10/11 (64-bit)
- **연결**: USB (모터 컨트롤러 연결용)

## 자동 업데이트

ADriveTool은 시작 시 자동으로 새 버전을 확인합니다. 업데이트가 있으면 알림이 표시됩니다.

## 문의

문제가 있거나 기능 요청이 있으시면 Issues 탭을 이용해 주세요.
